# Jornada Viagens

Projeto desenvolvido como treinamento de **HTML, CSS e responsividade**, utilizando como base um layout pronto no **Figma**. A proposta foi transformar o design em páginas web responsivas, respeitando a estrutura visual, os espaçamentos, as imagens e a adaptação para diferentes tamanhos de tela.

## Sobre o projeto

O projeto simula o site de uma agência fictícia chamada **Jornada Viagens**, especializada em pacotes de viagem, ofertas semanais, destinos populares e experiências turísticas.

O desenvolvimento teve como foco principal praticar a criação de interfaces responsivas a partir de um protótipo visual, trabalhando a adaptação do layout para mobile, tablet, desktop e telas maiores.

## Páginas desenvolvidas

O projeto conta com duas partes principais:

- **Página principal**, com apresentação da agência, ofertas da semana, categorias de pacotes, destinos populares, condições de pagamento, depoimentos e rodapé.
- **Página de pacotes de viagem**, criada como uma aba complementar do projeto, seguindo a mesma identidade visual e estrutura responsiva da página principal.

## Objetivos do treinamento

Este projeto foi criado com o objetivo de praticar:

- Estruturação semântica com HTML5;
- Estilização com CSS;
- Responsividade para diferentes tamanhos de tela;
- Uso de imagens adaptadas para mobile, tablet e desktop;
- Construção de header, hero, cards, seções de conteúdo e footer;
- Organização de classes CSS;
- Boas práticas de acessibilidade;
- Otimizações básicas de SEO;
- Fidelidade visual em relação ao layout do Figma.

## Tecnologias utilizadas

- HTML5
- CSS3
- Figma

## Recursos implementados

- Layout responsivo;
- Menu de navegação;
- Hero section;
- Cards de ofertas;
- Seção de categorias;
- Seção de destinos populares;
- Área de condições de pagamento;
- Depoimentos de clientes;
- Footer com informações de atendimento e redes sociais;
- Meta tags para SEO;
- Open Graph para compartilhamento em redes sociais;
- Dados estruturados com JSON-LD;
- Atributos de acessibilidade como `aria-label`, `aria-hidden` e `alt`.

## Acessibilidade e SEO

Durante o desenvolvimento, também foram aplicadas melhorias voltadas para acessibilidade e indexação, como:

- Link para pular diretamente ao conteúdo principal;
- Uso de títulos hierárquicos;
- Descrições alternativas em imagens relevantes;
- Elementos decorativos ocultos de leitores de tela;
- Meta description;
- Canonical URL;
- Tags Open Graph e Twitter Card;
- Estrutura JSON-LD para representar a agência fictícia.
